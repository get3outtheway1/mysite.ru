Make Header Sticky with One Click. Now you can also add scroll to top button with one click.

== Description ==

Easy installation one-click plugin for your WordPress website. To make your header sticky when scrolled, you need to add main header class(or id) into the plugin input. You can also add any color of your choice when the header is sticky.
There is another option where you can display scroll to top button in the front-end of the website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sticky-header-on-scroll.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open my-sticky-header page in the WordPress dashboard
4. Add your main header class or Id ( you can find your main header class by inspecting or press ctrl+shift+i in Google Chrome.
5. You can add any other color of your choice for header background when sticky.
6. With one-click you can add scroll to top button
